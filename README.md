# AI Automation & Web Scraping Tools

В этом репозитории собраны Python-скрипты, разработанные для автоматизации рутинных задач по сбору и обработке контента (ETL). Инструменты спроектированы с упором на отказоустойчивость, эвристический анализ данных и обход антибот-систем.

## 🛠 Содержимое

### 1. Article Image Downloader (`download_images.py`)
Умный парсер для скачивания контентных изображений из веб-статей.
* **Фильтрация мусора:** Анализирует DOM-дерево (BeautifulSoup) и отсеивает рекламу, аватары авторов, UI-иконки и пиксели отслеживания.
* **Обход защит:** Использует `curl_cffi` для имитации отпечатка браузера (impersonate Chrome) и успешного обхода Cloudflare.
* **Отказоустойчивость:** Валидация картинок через `Pillow`, ограничение размера, хеш-дедупликация (`hashlib`), обработка таймаутов.

### 2. News Aggregator & Translator (`news_scraper.py`)
Парсер новостей (GSMArena, PhoneArena) с автоматическим машинным переводом и выгрузкой данных.
* **Сбор данных:** Парсинг RSS-фидов и XML-sitemap (requests, BeautifulSoup).
* **Обработка:** Автоматический перевод заголовков и аннотаций с английского на русский (`deep_translator`).
* **Интеграция:** Бесшовный экспорт отфильтрованных и уникальных новостей в Google Таблицы по API (`gspread`, `OAuth2`).
* **Оптимизация:** Защита от дубликатов, автоматическая сортировка по дате и времени (МСК).

## 🚀 Стек технологий
* **Язык:** Python 3.8+
* **Библиотеки:** `BeautifulSoup4`, `curl_cffi`, `requests`, `gspread`, `Pillow`, `deep_translator`.
* **Подходы:** Web Scraping, DOM Parsing, ETL, LLM Prompting (скрипты спроектированы и отлажены с использованием продвинутого промпт-инжиниринга в Claude/Gemini).
